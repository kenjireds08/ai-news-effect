# 📰 Claude Code の週次リミットが 50% 増（7/13 まで）— 5月だけで3段階の緩和

> 一次情報: https://www.anthropic.com/news/higher-limits-spacex
> @ClaudeDevs 告知: https://x.com/ClaudeDevs/status/2054639777685934564
> 公開日: 2026-05-13

5月の Claude Code は **3 段階で枠が広がる流れ** になっています。

**5/6**: 5時間制限を **2 倍**（SpaceX Colossus 1 のキャパ獲得が背景）
**5/6**: Pro / Max のピークタイム制限を **撤廃**
**5/13**: 週次（7d）制限を **50% 増**（**7/13 までの期間限定**）

対象は **Pro / Max / Team / 座席ベース Enterprise**。Free プランは対象外。段階的ロールアウトなので、まだ自分の画面に反映されてなくても数日以内に来るはず。

## ちーけん的補足

未来AI学院 上級コース受講生で **来週からセットアップ予定の人にとっては、タイミングがかなり良い** です。導入期と演習期が緩和期間にまるごと重なるので、リミット詰まりで挫折する確率がだいぶ下がる。

ただし注意点が 2 つ。

ひとつは **7/13 で元のリミットに戻る**こと。コース最終週（7/14〜7/20）は通常モードに戻るので、最終週に大型の演習を当てている人は Plan モードで設計を固めてから一気に書くなど、ループを減らす運用を意識してください。

もうひとつは **Pro プランの場合は Sonnet 中心がオススメ**。Opus はトークン消費が大きく、Pro だと緩和されてもすぐ詰まりやすい。Sonnet 4.6 は十分高品質なので、課題演習レベルなら Sonnet で問題なし。Opus を使いたい局面は Max プラン以上が前提という設計感は変わっていません。

ちなみに、コードレビュー・要件定義の壁打ち・設計レビューなど「考えさせる作業」は Opus、実装の手数を稼ぐ作業は Sonnet、というモデル使い分けが今期の現実解です。
