# 📰 Claude Code の週次リミットが 50% 増（7/13 まで）— 5月だけで 3 段階の緩和

> 一次情報: https://www.anthropic.com/news/higher-limits-spacex
> @ClaudeDevs 告知: https://x.com/ClaudeDevs/status/2054639777685934564
> 公開日: 2026-05-13

5月の Claude Code は **3 段階で枠が広がる流れ** になっている。

**5/6**: 5時間制限を **2 倍**（SpaceX Colossus 1 のキャパ獲得が背景）
**5/6**: Pro / Max のピークタイム制限を **撤廃**
**5/13**: 週次（7d）制限を **50% 増**（**7/13 までの期間限定**）

対象は **Pro / Max / Team / 座席ベース Enterprise**。Free プランは対象外。未来AI学院 上級コース（5/18 開始 〜 7/20 終了）の **導入期と演習期が緩和期間に丸ごと重なる** タイミング。

⚠️ **注意**: 段階的ロールアウト中で、Max 契約者であるちーけん自身の環境でも **5/14 朝時点でまだ反映を確認できていない**。「公式は緩和済みと言ってるのに自分の画面では増えてない」と焦らず、数日待ってから判断するのが正解。
